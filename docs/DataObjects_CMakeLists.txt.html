Make a shared library containing the objects for all sources in this directory. You don’t have to list the sources, the build system will figure it out.  The library name should be prefixed with the package name (@gm2artexamples@ in this case).  You will have to use this library name in the directory with the modules. 


             		                            
cet_make( LIBRARY_NAME gm2artexamples_dataobjects )
